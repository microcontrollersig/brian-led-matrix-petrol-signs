# One Panel demos

Use minicom/mobaxterm as serial terminal.


The input format is:

{**command**}

Example:

{0}

Commands available

| Command |Description|
|---------|-----------|
|0|Top Left|
|1|Bottom right|
|2|Top right|
|3|Bottom left|
|4|Test Pattern 1|
|5|Snake Game|
|6|Single character|
|7|Static Text|
|8|Scrolling Text|
